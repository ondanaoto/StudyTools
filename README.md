# StudyTools
## Abstract
自分が普段勉強するのに利用しているツールをまとめています。
## Setup
1. このリポジトリをクローンします。
2. pythonをインストールしておきます。
3. `pip install -r requirements.txt`で必要なモジュールをインストールします。python環境を汚したくないので、仮想環境を作っておくことをおすすめします。
## ツール
### ポモドーロタイマー
#### 機能
一定の時間の間、作業開始のアラームと休憩開始のアラーム音を交互に鳴らします。
